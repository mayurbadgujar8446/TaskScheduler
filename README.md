# TaskScheduler
Developed a task scheduling application that uses Greedy and Dynamic Programming algorithms to optimize daily workloads. The system manages tasks, work schedules, and breaks, generates priority-based schedules, and provides reminder notifications to improve productivity.


⏰ AI Smart Task Scheduler

A web-based task scheduling system that intelligently optimizes daily workloads using Greedy and Dynamic Programming algorithms. The application helps users organize tasks, manage work schedules and breaks, maximize productivity, and generate optimized task plans based on available time and task priority.

🌟 Features

📝 Task Management
• Add and manage tasks with estimated completion time
• Assign priority levels to tasks
• Remove or update tasks easily

⏳ Work Schedule Planning
• Set working hours and break times
• Calculate available productive time automatically
• Support for multiple break intervals

🧠 Smart Optimization Algorithms
• Greedy Algorithm (Shortest Job First)
• Dynamic Programming (0/1 Knapsack)
• Maximizes task completion and priority score
• Generates optimized daily schedules

📊 Schedule Analysis
• Displays total available time
• Calculates time utilization
• Shows completed tasks and priority score
• Generates an optimized task timeline

🔔 Reminder System
• Browser notification reminders
• Automated alerts for scheduled tasks
• Real-time task tracking

💾 Data Persistence
• Stores tasks and schedules using Local Storage
• Preserves user data between sessions
• Fast client-side performance

🎨 User Interface
• Responsive and modern dashboard
• Interactive task management table
• Timeline visualization for schedules
• Clean and user-friendly design
